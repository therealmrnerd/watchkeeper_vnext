# EDSA Source Schematics

These SVG schematics were mirrored from https://siriuscorp.cc/edsa/ as source/reference assets for ships in the EDSA ship list, including ships missing from the local top-down schematic set.

They are kept separate from the runtime PNG schematics because the EDSA SVGs use a different visual style and coordinate system. They are useful for future conversion work and for extracting hardpoint metadata.
