Real-time cockpit and immersion platform for Elite Dangerous

WATCHKEEPER vNEXT

Your ship already produces the data. Watchkeeper gives it a voice.

Built around live telemetry, adaptive interfaces, automation, lighting integration, streaming systems and multi-display support, Watchkeeper transforms Elite Dangerous from a game on a monitor into a live command environment.

Open MFD Output Guided Layout Editor

This is not about cheating. This is not about automation playing the game for you. This is about immersion.

Elite Dangerous already generates an incredible amount of telemetry and environmental state. Watchkeeper takes that live in-game data and turns it into cockpit displays, contextual HUDs, adaptive interfaces, ambient lighting, stream integrations, operational panels and real-time situational awareness.

Your ship already knows
Fuel state Jump status Landing state SRV deployment Heat levels Cargo Missions Navigation System transitions Warnings Alerts Combat state Environment conditions

Built For Commanders

Watchkeeper is designed around the idea that every commander flies differently. Some want clean tactical overlays. Others want full cockpit immersion, dedicated touch displays, ship-state lighting, mission boards, external telemetry panels, Twitch/chat integration, AI-assisted workflows and multi-device bridge stations.

Watchkeeper supports both.

Not A Spreadsheet Overlay

Watchkeeper is designed to feel like part of the ship. The goal is not more windows. The goal is presence: a cockpit that reacts, a ship that communicates, and a bridge that feels alive.

Current Features
Real-time Elite Dangerous telemetry ingestion Adaptive cockpit UI Multi-pane contextual interfaces SRV / SLF / On-foot mode switching OBS integration SAMMI integration Dynamic lighting control Event-driven automation Policy-gated actions Multi-display architecture Drag-and-drop layout designer Modular panel system Live state persistence Streaming support External data integration

Philosophy

Most tools reduce telemetry into tables and efficiency stats. Watchkeeper goes the other direction and asks: what would an actual shipboard operations system feel like?

Roadmap

Full multi-display assignment system Shared layout profiles External telemetry providers AI advisory workflows Advanced mission integration Browser-rendered live panels Virtual display feeds Extended lighting ecosystems Voice/event orchestration Touchscreen-first layouts Streamer/operator split modes

Built By A Commander, For Commanders

Watchkeeper is being built as an active engineering project with real-world infrastructure, networking and operational design principles behind it. No buzzwords. No fake sci-fi fluff. Just building the cockpit we always wanted Elite Dangerous to have.
